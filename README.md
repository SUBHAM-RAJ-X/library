# Library Management System

A comprehensive full-stack library management system with React Native mobile app, Node.js backend, and Supabase database.

## Tech Stack

- **Frontend**: React Native (Expo)
- **Backend**: Node.js + Express
- **Database**: Supabase (PostgreSQL)
- **Storage**: Supabase Storage
- **Authentication**: Supabase Auth
- **Admin Panel**: React.js Web Dashboard

## Project Structure

```
library-management-system/
├── backend/                 # Node.js Express API
├── mobile-app/             # React Native Expo app
├── admin-dashboard/        # React.js web admin panel
├── supabase/              # Database schema and migrations
└── docs/                  # Documentation
```

## Features

- User authentication with role-based access control
- Book upload and download system
- Alphabetical storage organization (A-Z)
- Offline support with local caching
- Admin dashboard for management
- Search functionality
- Multi-user support with download tracking

## Getting Started

1. Set up Supabase project
2. Configure environment variables
3. Install dependencies
4. Run backend server
5. Start mobile app and admin dashboard
